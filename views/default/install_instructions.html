How to install Phylografter

  1. Install python, version 2.5 or higher
  2. Install web2py
  3. Start web2py, log in and navigate to the administration page
  4. Upload the phylografter appliance, giving it a name of your choice. The demo appliance ships with about 1,825 trees (mostly angiosperms) from TreeBASE as of early 2008.
