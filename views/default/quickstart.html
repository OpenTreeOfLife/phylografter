How to upload data

  1. Log in using an online account (Google, Yahoo!, etc.)
  2. Request "contributor" permission for your account by sending an e-mail to rree at fieldmuseum dot org.
  3. Create a new study record (published source of trees and data), by selecting an option from the "Studies" menu above:
    1. New study, or
    2. Import from TreeBASE
    (Note: you may want to search the studies table first to make sure the study has not already been added. E.g., search by typing the author's name into the text field at the bottom of the "Citation" column.)
  4. Upload trees and data associated with the study using the and buttons. If possible, upload a file containing the Genbank accession numbers of the sequences used in the study.
  5. Tag studies and trees with arbitrary terms using the "Tags" input field on the right side of their respective pages. For example, a tree might be tagged with the names of the genes used to reconstruct it.
  6. Map OTUs to taxa. Each study has an associated set of operational taxonomic units (OTUs) that should be mapped if possible to names in the reference taxonomy. Leaf nodes on uploaded trees with labels that match taxon names are mapped automatically. The others can be mapped manually:

    Clicking will bring up a table showing node labels in the first column and taxon names in the second. Click the name in the second column (will be "None" if the OTU is unmapped) and type the name of the correct taxon.

    Please note, at this time it is not possible to add new names to the reference taxonomy through Phylografter.
