Getting started

Phylografter is a web application for creating large phylogenetic trees "by hand."
The workflow is simple:
  1. {{=A("Log in", _href=URL(r=request,c="default",f="login"))}} (necessary to add, edit, and delete trees).
  2. Upload {{=A('"source trees"', _href=URL(r=request,c="stree",f="index"))}} (normally, these are from published literature).
  3. Create {{=A('"grafted trees"', _href=URL(r=request,c="gtree",f="index"))}} from source trees.
    1. Select a source tree to be the "backbone" of a new grafted tree.
    2. Select branches from other source trees. You can {{=A('search for taxa', _href=URL(r=request,c="taxa",f="index"))}} (named clades) to find branches of interest.
    3. Copy the branches to the clipboard.
    4. Graft them onto the grafted tree.
    5. Branches from grafted trees may be grafted onto other grafted trees.
